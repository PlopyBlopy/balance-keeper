# balance-keeper

Сервис управления финансовыми счетами с гарантированной доставкой событий через Kafka и паттерн Transactional Outbox. Демонстрирует подходы к построению надёжных распределённых систем на Go.

Технологии: Go, gRPC, REST, PostgreSQL, Kafka (franz-go), OpenTelemetry, Prometheus, Grafana, Loki, Jaeger, Docker.
